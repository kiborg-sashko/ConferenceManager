## Опис
Система призначена для управління науковими конференціями. 
Дозволяє обробляти інформацію про:
- Конференції (назва, дата, місце)
- Учасників
- Доповіді

## Структура проекту
1. **ConferenceManager.Core** - бібліотека класів (Domain Layer). Містить основні сутності.
2. **ConferenceManager.ConsoleApp** - консольний інтерфейс для демонстрації роботи.
3. **ConferenceManager.Tests** - Unit-тести (xUnit) для перевірки бізнес-логіки.

## Запуск
1. Клонувати репозиторій.
2. Виконати `dotnet run --project ConferenceManager.ConsoleApp`.
3. Для запуску тестів: `dotnet test`.
